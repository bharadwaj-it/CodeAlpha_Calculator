# C Calculator
This is a simple calculator program written in C programming language.

## Features
- Addition
- Subtraction
- Multiplication
- Division

## How to Run
1. Compile:
gcc calculator.c -o calculator

2. Run:
./calculator

## Sample Output
Enter first number: 10
Enter second number: 5
List of Operations
1.Addition
2.Subtraction
3.Multiplication
4.Division
Choose your choice: +  
Result of Addition operation: 15
